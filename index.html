Syntax VM

A powerful, low-level stack-based runtime built with Pascal.

  Download Latest Release   GitHub Repository

System Architecture

The VM operates on a custom execution model consisting of a Fetch-Decode-Execute loop, providing a bridge between high-level logic and native OS capabilities.

Stack-Based

All operations are performed on a LIFO stack. This includes arithmetic, logic, and function calls.

Bytecode Engine

Highly optimized execution of .syprg files with constant pool support.

Native FFI

Directly load Windows DLLs and call native functions with standard conventions.

File Specifications

Extension Type Description
.syasm Source Human-readable assembly mnemonics.
.syprg Binary Compiled bytecode for SVM execution.
